# Prettier-Config

## Usage 

```sh
yarn add @tenjojeremy/prettier-config
```
package.json
```json
{
  "prettier": "@tenjojeremy/prettier-config"
}
```

## Config 
```js
module.exports = {
  printWidth: 90,
  trailingComma: 'all',
  tabWidth: 2, 
  semi: false,
  singleQuote: true,
  jsxSingleQuote: true,
  arrowParens: 'always',
}
```
